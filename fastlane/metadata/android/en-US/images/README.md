# F-Droid / Fastlane Image Assets

Place the following image files in this directory:

- `icon.png` -- 512x512 app icon
- `featureGraphic.png` -- 1024x500 feature graphic (displayed at top of F-Droid listing)
- `phoneScreenshots/` -- phone screenshots (at least one recommended, named `1.png`, `2.png`, etc.)

See https://f-droid.org/docs/All_About_Descriptions_Graphics_and_Screenshots/ for full requirements.
